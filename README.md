# CV Interactif - Déguéne Dieng

## Description

Ce projet est un curriculum vitae interactif développé dans le cadre du module de Développement Web en Licence 2 Informatique à l'Université Iba Der Thiam de Thiès.

L'objectif est de présenter mon parcours académique, mes compétences et mes projets à travers une page web moderne, responsive et interactive.

## Technologies utilisées

 HTML5
 CSS3
 JavaScript 
 LocalStorage

## Fonctionnalités

 Navigation entre les différentes sections du CV.
 Design responsive adapté aux ordinateurs et aux téléphones.
 Mode sombre / mode clair avec sauvegarde dans le LocalStorage.
 Menu hamburger pour les petits écrans.
 Animation des barres de compétences.
 Formulaire de contact avec validation des champs.
 Bouton « Retour en haut ».

 Ouvrir le dossier du projet.
 Double-cliquer sur le fichier "index.html" ou l'ouvrir dans un navigateur web.
